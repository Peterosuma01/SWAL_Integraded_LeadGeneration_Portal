# Steelwool Africa LTD - Integrated Lead Generation System

A unified PWA for managing Hardware and Homecare leads.

## Features
- 🔩 Hardware Lead Generation
- 🏠 Homecare Lead Generation
- 👤 Sales Representative Login
- 👔 Supervisor Dashboard
- 📊 Lead Tracking & Management
- 📈 Orders Management
- 📱 PWA Support

## Deployment
This app is deployed on GitHub Pages and uses Google Apps Script as the backend.

## Setup
1. Deploy the Google Apps Script backend
2. Update the API_URL in index.html
3. Push to GitHub
4. Enable GitHub Pages

## Technologies
- HTML/CSS/JavaScript
- Google Apps Script
- Google Sheets
- PWA (Service Worker + Manifest)
- jsPDF & SheetJS for exports
